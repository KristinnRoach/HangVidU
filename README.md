# HangVidU

Peer-to-peer video chat app with text messaging and synced media sharing capabilities.

## Try it now

**Live demo**: [https://hangvidu.com/](https://hangvidu.com/)

## What it does

- 1-to-1 video calls over WebRTC
- Invite friends via referral link or directly if account exists
- Instant persistent messaging with attachment storage
- Push notifications for calls and messages
- Synced media playback between contacts
